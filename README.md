# TileVania
Welcome to TileVania, a 2D platformer game inspired by classic Mario games. Developed using C# and the Unity game engine, this project takes advantage of Unityâ€™s Tilemap tool to create immersive levels and environments. - Seamlessly integrated graphics and assets.
- Continuous playtesting and optimization for smooth performance.
